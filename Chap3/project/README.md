# 内网版天气查询程序
*****
# Start
## 1. 在命令行中运行python文件
```
python weather_web.py
```
## 2. 复制命令行中显示的url黏贴到浏览器中
http://127.0.0.1:5000/
## 3. 输入城市名，查询对应天气
***
# command
## 1. search 
### 在输入框中直接输入要查询的城市名，点击‘search’查询对应天气
## 2. help 
### 点击‘help’，获得用户帮助说明
## 3. history
### 点击‘history’，查看查询过的所有城市
